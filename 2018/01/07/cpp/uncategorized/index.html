名字空间$namespace$

  • 名字空间将全局作用域细分为独立的, 具名的作用域, 可有效防止全局作用域的命名冲突。
  • 在头文件中使用匿名空间导致违背 C++ 的唯一定义原则 (One Definition Rule (ODR))。

匿名名字空间

在$cpp$文件中, 允许甚至鼓励使用匿名名字空间, 以避免运行时的命名冲突。
